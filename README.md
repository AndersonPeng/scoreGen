# scoreGen

Score generator for estimating notes by using FFT, HPS, deep neural network and autoccorelation

#### Prerequisite
* Python 3.4.3 (x64)
* PyQt 5.4.1 for Python3.4 (x64)
* numpy
* pyaudio

#### Execution
  python main.py
  
#### Building an executable file with cx_freeze
  python setup.py build
